1. 不建议按照 [官网安装文档](https://docs.flutter.dev/get-started/install)
   直接下载解压安装。
2. 建议直接克隆(clone)其 [GitHub源码仓库](https://github.com/flutter/flutter)。
3. 克隆时可选择指定分支，例如`stable`(稳定发行版)。
4. 直接从GitHub克隆网速可能受到限制，推荐使用镜像。
